//***authentification***//

@startuml
activate Administrateur
Administrateur -> Système: 1: Saisir le login et le mot de passe
activate Système
Administrateur -> Système: 2: Validation
Système -> SGBD :2.1: Envoyer
activate SGBD
SGBD -> SGBD: 3: Vérifie
alt             login ou mot de passe incorrecte
    
    SGBD --> Système:4: informations incorrecte
    Système --> Administrateur:5: Réaffichage du formulaire d'authentification avec un message d'erreur
    
else login et mot passe correcte
    SGBD --> Système:6:informations correctes
    Système --> Administrateur: Affichage de l'espace correspondant
    
end
@enduml

//***ajouter un produit***//

@startuml
activate Administrateur
Administrateur -> Système: 1: Demande d'ajouter un produit
activate Système
Système --> Administrateur :1.1: Affichage du formulaire d'ajout
Administrateur -> Système: 2: Remplir le formulaire
Système -> Système: 2.1: Vérification du remplissage

alt             Remplissage correcte
activate SGBD
Système -> SGBD :3: Envoyer les informations 
SGBD -> SGBD: 3.1: Vérification d'existance

    
alt     le produit n'existe pas
               
SGBD --> Système: 4: Confirmation d'ajout
Système --> Administrateur :4.1: Confirmation d'ajout du produit
    
else    Sinon
SGBD --> Système: 5: Le produit existe
Système --> Administrateur :5.1: Le produit existe déjà
end
    SGBD --> Système:4: informations incorrecte
    Système --> Administrateur:5: Réaffichage du formulaire d'authentification avec un message d'erreur
    
else Sinon
    Système --> Administrateur:6: Message d'erreur
    
end
@enduml

//***modifier un produit***//

@startuml
activate Administrateur
Administrateur -> Système: 1: Demander la liste des produit
activate Système
Système -> SGBD : 2: Envoyer la demande
SGBD -> SGBD : 3: Chargement 
SGBD --> Système:4 : Réponse
Système --> Administrateur: 4.1: Afficher la liste des produits
Administrateur -> Système: 5: Selectionner un produit
Administrateur -> Système: 6: Demander de modifier un produit
Système --> Administrateur: 7: Afficher le formulaire
Administrateur -> Système: 8: Saisir les information puis valider
Système ->Système: 8.1: Vérification
    
alt     Champ incomplet
               
Système --> Administrateur : 9: Afficher le message d'erreur
else                     Champ Complet
Système -> SGBD: 10: Envoyer la demande
SGBD -> SGBD : 10.1: Enregistrer
SGBD --> Système : 11: Réponse
Système --> Administrateur :5.1: Afficher le message de confirmation
end

@enduml
